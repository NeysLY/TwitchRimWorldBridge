# Twitch RimWorld Bridge

Приложение-сервер для мода **Twitch RimWorld Bridge**, позволяющее зрителям Twitch влиять на игру в RimWorld в реальном времени.

## 📥 Скачать

**[⬇️ Скачать последний релиз](https://github.com/neysly/TwitchRimWorldBridge/releases/latest)**

&gt; ⚠️ **Важно:** Это приложение является обязательным дополнением к моду. Без него мод не будет работать.

## 🚀 Быстрый старт

1. Скачай последний релиз (ZIP-архив) по ссылке выше
2. Распакуй архив СТРОГО в корневую папку **"Steam\steamapps\common\"**
3. В игре зайди в настройки мода (F12 → Twitch Bridge) и нажми **"Запустить"**
4. Нажми **"Подключить"** — индикатор станет зелёным ✅

## 🎮 Возможности

- Зрители появляются в игре как колонисты
- Голосование за события, рейды, подарки
- Отправка предметов, животных, ресурсов
- Кастомные эффекты через баллы канала

ПОДДЕРЖКА ANTHROSONAE
═══════════════════════════════════════

Мод полностью совместим с Anthrosonae!
Зрители, присоединяющиеся к колонии, могут появиться как антропоморфные персонажи с уникальными расами, хвостами и ушками. Все кастомные черты и морфологии из Anthrosonae сохраняются при генерации колонистов из чата.

## 📋 Требования

- Windows 10/11
- .NET 8.0 Runtime (или выше)
- Мод [Twitch RimWorld Bridge](https://steamcommunity.com/sharedfiles/filedetails/?id=XXXXXXXXX) в Steam Workshop
