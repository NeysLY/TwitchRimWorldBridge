# 🎮 Twitch RimWorld Bridge

Интеграция Twitch-чата с RimWorld. Зрители влияют на колонию в реальном времени!

> ⚠️ **Не доступно в Steam Workshop** из-за ограничений платформы.  
> Скачивай только с GitHub Releases.

---

## 📥 Скачать

**[➡️ Последний релиз](https://github.com/neysly/TwitchRimWorldBridge/releases/latest)**

Файл: `RimWorld.zip`

---

## 🚀 Установка

1. Скачай `RimWorld.zip`
2. Распакуй архив, указав путь к папке Steam: **"Steam\steamapps\common\"**
3. Архив сам создаст нужные папки:
- `RimWorld\Mods\TwitchRimWorldBridge\` — мод
- `RimWorld\Mods\TwitchRimWorldBridge\app` — приложение

3. В лаунчере RimWorld включи мод **Twitch RimWorld Bridge** (после HugsLib)

4. Запусти игру, дождись 
5. Нажми **F12 → Twitch Bridge → "Запустить" 
"Доздись загрузки приложения" → "Нажми на запустить сервер" → "Подключить в приложении"**

6. Индикатор зелёный ✅ — готово!

## 🎮 Возможности

- 👤 Зрители появляются как колонисты
- ⚔️ Голосование за события, рейды, подарки
- 📦 Отправка предметов, животных, ресурсов
- 🐺 **Полная поддержка Anthrosonae** — фурри колонисты из чата

## 📋 Требования

- RimWorld 1.6
- [HugsLib](https://steamcommunity.com/sharedfiles/filedetails/?id=818773962) (из Steam Workshop)
- Windows 10/11
- .NET Framework 4.8 (мод) / .NET 8.0 (приложение)
