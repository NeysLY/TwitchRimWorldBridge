# Twitch RimWorld Bridge

Приложение-сервер для мода **Twitch RimWorld Bridge**, позволяющее зрителям Twitch влиять на игру в RimWorld в реальном времени.

## 📥 Скачать

**[⬇️ Скачать последний релиз](https://github.com/neysly/TwitchRimWorldBridge/releases/latest)**

&gt; ⚠️ **Важно:** Это приложение является обязательным дополнением к моду. Без него мод не будет работать.

## 🚀 Быстрый старт

1. Скачай последний релиз (ZIP-архив) по ссылке выше
2. **Распакуй архив СТРОГО в корневую папку игры __\steamapps\common\RimWorld\__**
